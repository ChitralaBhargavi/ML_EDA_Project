## Objective
To build a Machine Learning model to predict whether a person has diabetes based on medical parameters.

## Dataset
The diabetes dataset contains features such as glucose level, blood pressure, BMI, age, etc.

## Steps Performed
- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Machine Learning model training
- Model evaluation

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Conclusion
The model predicts diabetes using medical data and demonstrates the application of Machine Learning in healthcare.
